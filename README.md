# BuildSmart — Home Construction Cost Estimator

An AI-powered, interactive web application designed to help homeowners in India estimate construction costs with high accuracy.

## 🚀 Features
- **Smart Estimation**: Rule-based engine with AI refinement (Llama 3.3).
- **Location Aware**: 20+ major Indian cities supported + Smart AI fallback for unlisted locations.
- **Material Quantity Estimator**: Get approximate counts of Cement, Steel, Bricks, etc.
- **EMI Calculator**: Built-in financial planning tool.
- **Interactive Chatbot**: "BuildSmart Assistant" to help with construction queries.
- **Professional Reports**: Download estimates as branded PDF or PNG.
- **Dual Unit Support**: Input area in Square Feet or traditional Gunthas.

## 🛠️ Tech Stack
- **Frontend**: React (Vite), CSS Modules, Chart.js, html2canvas.
- **Backend**: Python (Flask), Groq AI (Llama 3.3).
- **Styling**: Vanilla CSS with modern Glassmorphism aesthetics.

## 📦 Setup & Installation

### Backend
1. Navigate to `/backend`
2. Create a virtual environment: `python -m venv venv`
3. Install dependencies: `pip install -r requirements.txt`
4. Add your `GROQ_API_KEY` to `.env`
5. Run: `python app.py`

### Frontend
1. Navigate to `/frontend`
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
